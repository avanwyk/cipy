# CIPy
Computational Intelligence algorithms in Python using NumPy.

### Installation
* Source (development)

    ```shell
    git clone https://github.com/avanwyk/cipy
    cd cipy
    pip install -e .
    python examples/gbest_pso.py
    ```
* Source (setup.py)

    ```shell
    git clone https://github.com/avanwyk/cipy
    cd cipy
    python setup.py install
    python examples/gbest_pso.py
    ```
* PyPI

    ```shell
    pip install cipy
    ```
